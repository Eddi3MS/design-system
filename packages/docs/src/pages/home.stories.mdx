# Design System - Rocketseat Ignite

- Lib of Components madeup with <b>Radix-UI</b> and styled with <b>Stitches</b>.

## Storybook

- In the tokens section, you'll find all the spaces, fonts, colors, all the variables used in my components.

- Note: you can override everything just by using your own variables inside the css property of a component.

```
<Component css={{fontSize: '75px'}}/>
```

- Right after the tokens section, You'll find an example of each component made.
